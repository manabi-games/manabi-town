# 画像生成AI用 素材依頼プロンプト集

「おべんきょコレクション」のキャラ素材を画像生成AIに依頼するときのプロンプト。

## 依頼のしかた（重要）

1. **必ず添付する画像**: `base_character.png`（髪なしの基準キャラ。頭の位置・サイズ合わせ用）
   - 画風の参考に `style_reference.png` も一緒に添付するとより安定します
2. **透過PNG（背景透明）対応のAI**で依頼する（ChatGPT/GPT-4o、Gemini など）。
   透過が無理な場合は「完全に真っ白な背景」で依頼（こちらで白を抜くが、輪郭に白フチが残ることあり）
3. いきなり全部頼まず、**まず①髪型シート1枚だけ**試して品質を見るのがおすすめ
4. できた画像は `claudeゲーム開発` フォルダに保存して、Claudeに「素材できたよ」と知らせる
   → Claudeが切り抜き・位置調整してゲームに組み込みます

## ① 髪型シート（10種）

```
添付画像は、子供向けブラウザゲームの主人公キャラです（正面向き・フラットなベクター風イラスト）。
このキャラの丸い頭にぴったりかぶせられる「髪型パーツだけ」を10種類、1枚のシートにまとめて描いてください。

【仕様】
- 出力: 2000×1000ピクセル、背景は完全に透明の透過PNG
- 5列×2行のグリッドに等間隔で配置（1マス400×500ピクセル、パーツは各マスの中央）
- 10種すべて、添付キャラの頭と同じ大きさ・同じ正面向きで統一すること（あとでそのまま重ねるため）
- 顔・体は描かない。髪の毛パーツのみ

【髪型の内容】
上段（女の子向け）: おだんごヘア／みつあみ2本／前髪ぱっつんボブ／ロングウェーブ／サイドポニーテール
下段（男の子向け）: マッシュヘア／ソフトモヒカン／くせ毛ショート／ヒーロー風ツンツンヘア／アフロ

【スタイル】
- 添付イラストと同じ、フラットでシンプルな塗り（線なし・ベタ塗り＋ハイライト1段まで）
- 髪色はぜんぶ「こげ茶色（#4a3728）」で統一（ゲーム側で色を変えるため）
- 文字・数字・枠線・影・背景は一切入れない
```

## 受け取り後の組み込みメモ（Claude用）

- シートは PowerShell + System.Drawing でグリッド切り抜き → `assets/parts/` に配置
- 髪・帽子・服は `avatar.js` に PNG レイヤーとして追加（SVGパーツと共存、`<image>`要素で重ねる）
- 位置ずれは1パーツずつ offset 調整（data.js に {dx, dy, scale} を持たせる）
- 透過でない素材が来たら白背景除去を試みる（フチ残りに注意）
