# まなびタウン 🏘️

小学1年生向けの「トモダチコレクション風」学習ゲームです。
勉強レベルをクリアするとコインがもらえて、服・ぼうし・めがね・部屋のかざりを買ったり、
まちにトモダチやペットが増えていきます。オンライン要素は一切ありません。

## あそびかた

1. さいしょに じぶんのキャラを つくる（はだ・かみがた・め・ふくのいろ）
2. 「べんきょう」から レベル1〜30に ちょうせん（1レベル5もん・3たくしき）
3. 3もん いじょう せいかいで クリア！ コインと ⭐がもらえる
4. コインで おみせの アイテムを かって きせかえ
5. レベルが すすむと まちに トモダチ（6にん）と ペット（5ひき）が やってくる

## 学習内容（難易度カーブ）

| レベル | 内容 |
|---|---|
| 1–5   | かぞえる・ひらがな・おおいすくない |
| 6–10  | たしざん（〜10）・カタカナ |
| 11–15 | ひきざん（〜10）・かんじのかず |
| 16–20 | 1年生のかんじ・くりあがりのたしざん・とけい（〜じ） |
| 21–25 | くりさがりのひきざん・とけい（〜じはん）・100までのかず |
| 26–30 | おおきいかず・ぶんしょうだい・まとめテスト |

問題はすべて自動生成（単語・漢字はバンクからランダム選出）なので、何度あそんでも違う問題が出ます。
🔊ボタンで問題文を読み上げます（ブラウザの音声合成を使用）。

## 技術構成

- HTML / CSS / Vanilla JS のみ（ビルド不要・サーバー不要の静的サイト）
- セーブデータは `localStorage`（端末内のみ）
- アバターはSVGで動的生成（`js/avatar.js`）
- 問題ジェネレーター: `js/questions.js` / データ定義: `js/data.js` / 画面ロジック: `js/main.js`

## GitHub Pages で公開する手順

```bash
cd manabi-town
git init
git add .
git commit -m "まなびタウン v1"
# GitHubで manabi-games/manabi-town リポジトリを作成してから:
git remote add origin https://github.com/manabi-games/manabi-town.git
git branch -M main
git push -u origin main
```

その後、GitHubのリポジトリ設定 → Pages → Branch を `main` / `(root)` にして保存。
数分後に `https://manabi-games.github.io/manabi-town/` で遊べるようになります。

## ローカルで動かす

`index.html` をブラウザで開くだけでも動きます（Googleフォント以外はオフラインでも動作）。
