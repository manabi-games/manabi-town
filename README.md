# おべんきょコレクション 🏘️

小学1年生向けの「トモダチコレクション風」学習ゲームです。
横スクロールの まちを 自分のキャラで歩きまわり、たてものに入って あそびます。
勉強をクリアするとコインがもらえて、きせかえアイテムを買ったり、
まちにトモダチやペットが増えていきます。オンライン要素は一切ありません。

公開URL: https://manabi-games.github.io/manabi-town/

## まちの たてもの

| たてもの | なかみ |
|---|---|
| 🏠 じぶんのいえ | じぶんの部屋。きせかえ（髪型18種・帽子18種・おようふく10種ほか）・家具のかざりつけ |
| 🏫 がっこう | 3択クイズの30レベル（かず・ひらがな・カタカナ・かんじ・とけい・けいさん・文章題） |
| ⌨️ タイピングどうじょう | ローマ字20レベル＋けいさん20レベル。キーボードで答えを打つ（つりじまゲームの問題を移植） |
| 🛍️ おみせ | ぼうし・めがね・ふく・もちもの・へやのかざり |
| 🏡 トモダチのいえ ×6 | がっこうのレベルが進むと引っ越してくる。家に上がっておしゃべり |
| ⛲ こうえん | ペット（5匹）があつまる |

## 操作

- **← →キー / 画面のボタン / 地面タップ** で歩く
- たてものをタップ（または前で**Enter**）で入る
- タイピングは **Enterで回答 → Enterで次へ**（大文字小文字どちらでもOK、si/shi など別の打ち方も正解）

## 学習内容（難易度カーブ）

- がっこう Lv1–30: かぞえる → たしざん・カタカナ → ひきざん → 漢字・くり上がり・とけい → くり下がり・100までの数 → 文章題・まとめ（小1の1年間相当）
- タイピング ローマ字 Lv1–20: A I U E O → か行〜わ行 → ことば → しゃ・ちゃ → っ → ん → 長いことば
- タイピング けいさん Lv1–20: 数字うち → たし算 → ひき算 → くり上がり → 2桁の計算

がっこうの問題は自動生成なので毎回変わります。🔊ボタンで問題文を読み上げます。

## 技術構成

- HTML / CSS / Vanilla JS のみ（ビルド不要の静的サイト）
- セーブは `localStorage`（端末内のみ）
- アバター・建物はSVGで動的生成、まちは requestAnimationFrame の横スクロール
- `js/data.js` データ定義 / `js/questions.js` 3択問題生成 / `js/typing.js` タイピング / `js/town.js` まちあるき / `js/main.js` 画面ロジック / `js/avatar.js` アバター描画

## ローカルで動かす

`index.html` をブラウザで開くだけで動きます。
